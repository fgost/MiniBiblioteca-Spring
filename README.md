# MiniBiblioteca-Spring

Projeto que permite o usuario a criar autores autor e livros 

  -inserir livros do autor
  
  -modificar
  
  -listar
  
  -apagar(Quando o autor é apagado, os livros vinculados a ele também são.)
  

Tecnologias utilizadas:

  -Java 11
  
  -Spring Boot
  
  -Thymeleaf
  
  -CSS(Bootstrap)
  
  -HTML
